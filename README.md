# Delaware Pepper Co. Website

A redesigned website for Delaware Pepper Co. featuring a bold, modern aesthetic that captures the craft authenticity of small-batch hot sauce production.

## Files Included

- **index.html** - Homepage with hero section, featured products, brand story, and newsletter signup
- **products.html** - Complete product catalog with all pepper flakes (pure and blended)

## Features

### Design Elements
- Custom typography using Anton, Bebas Neue, and Crimson Pro
- Animated grain texture overlay
- Custom cursor with hover effects
- Heat-inspired color palette (deep reds, burnt orange, charcoal, cream)
- Scroll-triggered animations
- Responsive design for mobile and desktop

### Homepage (index.html)
- Full-screen hero with animated flame indicators
- Featured products section with heat level badges
- Brand story section
- Newsletter signup form
- Navigation to products page

### Products Page (products.html)
- Complete catalog of Pure Pepper Flakes:
  - Big Bang Chocolate Naga
  - Arse'olova Jolokia
  - Bird (Crispy Tepin)
  - Goat's Weed
  - Fish
  
- Complete catalog of Blended Flakes:
  - BBCN x Goat
  - BBCN x Fish
  - Arse x Bird
  - Bird x Goat

- SHU (Scoville Heat Unit) ratings
- Heat level indicators (Hot/Medium/Mild)
- Flavor profiles
- Contact information for ordering

### Navigation
Both pages feature consistent navigation with smooth scrolling and hover effects. Pages are linked together for seamless browsing.

## Usage

1. Open **index.html** to start at the homepage
2. Navigate to products via the navigation menu or "View All Products" button
3. All interactive elements (buttons, links) have hover effects
4. Contact email: Htam@delawarepepperco.com

## Customization Notes

- All colors are defined as CSS variables at the top of each file for easy customization
- Images are placeholder gradients - replace with actual product photography
- Newsletter form currently shows alert - integrate with your email service provider
- Social media links are placeholders - update with actual URLs

## Browser Compatibility

Works best in modern browsers (Chrome, Firefox, Safari, Edge). Custom cursor is disabled on mobile devices for better touch experience.
